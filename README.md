# Coach · Fitness

Web app leggera (singolo file HTML, offline) che genera allenamenti su misura in base a obiettivo, tempo, luogo e attrezzatura — con diario/progressione, timer e nutrizione.

- **index.html** — pagina d'ingresso
- **app.html** — profilo principale
- **app-lei.html** — profilo donna (dati separati)

I dati (diario, peso, macro) restano sul dispositivo (localStorage). Da *Diario → Esporta* si salva un backup su file.
